# constfilt

A header-only C++17 library for designing IIR digital filters at compile time.
Filter coefficients are computed via `constexpr` from filter specifications — no
offline tools (MATLAB, Python, etc.) required, and no processor time spent on
coefficient computation at runtime.

Built on [consteig](https://github.com/mitchellthompkins/consteig) for all
compile-time math (eigendecomposition, LU solve, matrix exponential).

## What It Does

All at compile time, constfilt supports:

- **ZOH discretization** of continuous-time state-space models via matrix
  exponential (`expm`) computed through eigendecomposition.
- **Butterworth lowpass filters**, orders 1–8, from cutoff frequency and sample
  rate alone.
- **Direct Form II Transposed** filtering — both a real-time sample-by-sample
  path and a `constexpr`-capable batch path.

## Quick Start

```cpp
#include "constfilt.hpp"

// All coefficient computation happens at compile time.
static constexpr constfilt::Butterworth<double, 2> filt(100.0, 1000.0);
//                                              order ^    fc ^    fs ^

// Compile-time batch filtering:
static constexpr consteig::Array<double, 32> input = { /* ... */ };
static constexpr auto output = filt(input);

// Real-time sample-by-sample (mutates internal state):
constfilt::Butterworth<double, 2> rt_filt(100.0, 1000.0);
double y = rt_filt(x);
```

`constfilt` is header-only. Add it to your project with `#include "constfilt.hpp"`.
Requires C++17 and a path to `consteig/` as a sibling directory.

## CMake

```cmake
add_subdirectory(constfilt)
target_link_libraries(your_target constfilt)
```

constfilt pulls in consteig automatically via `add_subdirectory(../consteig ...)`.

## Building and Testing

Tests run inside the same dev container as consteig. From the `constfilt/` directory:

```sh
# GCC
make test.gcc

# Clang
make test.clang

# Start an interactive container shell
make container.start
```

The container image is pulled automatically on first use.

## Options

| Macro | Default | Description |
|---|---|---|
| `CONSTFILT_PI` | `3.14159...` | Value of π used in frequency conversions. Override before including `constfilt.hpp`. |

## Supported Filters

| Filter | Orders | Discretization |
|---|---|---|
| Butterworth lowpass | 1–8 | ZOH |

## Tolerances (vs. Octave reference)

| Quantity | Tolerance |
|---|---|
| Transfer function coefficients | < 1e-9 |
| Step response (32 samples) | < 1e-7 |

Reference values are generated by `octave/generate_butterworth_tests.m` using
Octave's `c2d(..., 'zoh')` and committed to `tests/butterworth_reference.hpp`.
